# 🛒 Product Recommendation System using Apriori Algorithm

## 📌 Overview
This project demonstrates a simple product recommendation system using the Apriori Algorithm and Association Rule Mining. It identifies frequently purchased items and generates association rules to recommend complementary products.

## 🚀 Features
- Uses the Apriori algorithm to find frequent itemsets.
- Generates association rules based on confidence.
- Displays support, confidence, and lift for each rule.
- Built using Python, Pandas, and mlxtend.

## 🛠️ Technologies Used
- Python
- Pandas
- mlxtend

## 📂 Dataset
A sample dataset containing transactions of blue shirt products is used for demonstration.
